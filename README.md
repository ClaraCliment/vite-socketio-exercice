# Premier exercice avec Vite & Socket.io
</br>
- Simple connexion d'un serveur à un client grâce à socket.io
</br>
- Mise en place d'un client React avec Vite
</br>
</br>

### Lancer le projet:
---
Dans un premier terminal:
</br>
`cd client`
</br>
`npm install`
</br>
`npm run dev`
</br>
Dans un deuxième terminal:
</br>
`cd server`
</br>
`npm install`
</br>
`npm start`
